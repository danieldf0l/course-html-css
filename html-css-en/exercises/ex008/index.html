The main formats

Bold X Strong

In HTML4 (discontinued): using the tag <b>

In HTML5 (non-discontinued): using the tag <strong>.

Italic X emphasis

In HTML4 (discontinued): using the tag <i>.

In HTML5 (non-discontinued): using the tag <em>.

Insert X Underline

In HTML4 (discontinued): using a tag <u>.

In HTML5 (non-discontinued): using <ins>.

A bigger and a smaller text

In HTML4 (discontinued): using the tag <big>.

In HTML5 (non-discontinued): using the tag <small>.

Highlighter

In HTML5 (non-discontinued): using the tag <mark>.

Deleted text

In HTML5 (non-discontinued): using a tag <del>.

Superscript text

In HTML5 (non-discontinued): using a tag <sup>.

Subscript text

In HTML5 (non-discontinued): using a tag <sub>.
